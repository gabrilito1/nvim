**Environment**  
   - **Node.js e npm** (for `coc.nvim`):  
     ```bash
     sudo apt install nodejs npm
     ```
   - **xclip** (for clipboard on linux):  
     ```bash
     sudo apt install xclip
     ```
   - **ripgrep** and **fd** (for`telescope.nvim`):  
     ```bash
     sudo apt install ripgrep fd-find
     ```
   - **C++ Compilator** (Bind == `<F5>`):  
     ```bash
     sudo apt install g++
